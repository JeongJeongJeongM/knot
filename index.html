KNOT
AI 대화 행동 분석

KNOT이란?

KNOT은 AI 챗봇과의 대화 기록을 분석하여, 사용자의 대화 행동 패턴을 다각도로 시각화하는 도구입니다.

11개의 행동 축을 기반으로 대화 스타일을 프로파일링하고, 10가지 행동 유형 중 가장 가까운 패턴을 매칭합니다. 두 사람의 결과를 비교하여 호환성 분석도 가능합니다.

🔒 개인정보 안내
• 업로드된 대화 데이터는 브라우저에서만 처리됩니다
• 분석을 위해 Anthropic Claude API로 암호화된 연결(HTTPS)을 통해 전송됩니다
• Anthropic API 정책에 따라 전송된 데이터는 모델 학습에 사용되지 않습니다
• 분석 완료 후 원본 데이터는 브라우저 메모리에서 즉시 삭제됩니다
• 공유 기능 사용 시 분석 결과(점수/유형)만 저장되며, 원본 대화는 포함되지 않습니다
• 저장된 공유 데이터는 30일 후 자동 삭제됩니다
